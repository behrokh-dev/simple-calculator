# 🧮 Simple Calculator

A beginner-friendly calculator built with **HTML, CSS, and JavaScript**.  
This project was created as one of my first JavaScript exercises to practice DOM manipulation, event handling, and implementing basic arithmetic operations.

## 🚀 Live Demo

🔗 https://behrokh-dev.github.io/simple-calculator/

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear functionality
- 📱 Responsive design
- ⚡ Fast and lightweight
- 🎯 Simple and user-friendly interface

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
